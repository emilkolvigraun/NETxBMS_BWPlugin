Welcome

This the BossWave plugin for NETx BMS Studio.
The project can be found on GitHub.

Instructions

  1. Set your identity by clicking the tab 'BossWave Identity'
    1. Set the path to your entity.

  2. Now you can either publish or subscribe to a namespace by clicking 'Add'

    1. Publish
      1. Fill out all input fields.
      2. You can check off 'loop' and the plugin will publish the message with the declared interval.
        Default is 5000ms.
    2. Subscribe
      1. Fill out all input fields.
      2. It is not nessecary to declare an expiry or an expiry delta, however, it is recommended.
      3. If you check off 'auto chain' the plugin will find and use the shortest chain in the wave and catch it.
      4. Expiries are in miliseconds.
